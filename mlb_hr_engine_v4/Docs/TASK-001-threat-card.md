\# TASK-001 — Tactical HR Threat Card



\## Objective

Build a reusable tactical HR Threat Card component for MLB HR ENGINE.



\## Purpose

The component should visually communicate:

\- HR probability

\- barrel danger

\- pitch vulnerability

\- weather amplification

\- escalation status



The card should feel cinematic, operational, and intelligence-driven.



\---



\## Requirements



\### Data Points

\- Player Name

\- Team

\- HR Threat Score

\- Barrel %

\- Hard Hit %

\- Pitch Matchup Edge

\- Weather Boost

\- Pitcher Vulnerability

\- Escalation Tier



\---



\## UI Doctrine

\- cinematic tactical HUD

\- restrained glow

\- layered hierarchy

\- premium realism

\- modular layout

\- compact intelligence density



Avoid:

\- sportsbook styling

\- fantasy sports aesthetics

\- oversized whitespace

\- cartoon visuals



\---



\## Technical Requirements

\- React component

\- TypeScript typed props

\- Tailwind styling

\- reusable architecture

\- responsive layout



\---



\## Deliverables

\- reusable component

\- mock data example

\- clean prop interface

\- modular styling

\- documentation update
