# TAB SURVIVAL RULES

A permanent top-level tab must:

* provide unique intelligence
* have high revisit value
* materially change decisions
* provide fast scan value
* avoid major overlap with existing tabs

* an expandable module
* a tactical overlay
* a secondary intelligence panel
* a player detail system
* a portfolio module

…then it should NOT become a permanent top-level tab.
